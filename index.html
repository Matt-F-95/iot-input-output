      Concept of Project
    • The overall concept of this project was to create a multifunctional tool that would allow a user to view either the temperature or compass
    • This project works by use of separate functions that run whenever the A or B button is clicked on the micro bit board.
      On-start & Forever Functions
    • On start-up, the board it plays a small little melody and shows a ‘Welcome’ message
    • Afterwards, a function runs that tells the user to hit the A button to view temperature and shows and arrow to the button, then shows an arrow to the B button along with a message telling users to hit this button to view compass location.
      • A button allows the users to see the temperature when it is pressed
    • As well, there is an if else statement that runs if the temp is below 21°C it sends a sad face along with a tiny melody that is played. In addition, when the temperature is at or above 21°C it sends out a happy face and a ‘ode to joy’ melody is played.
      • B Button allows the users to see your compass location when it is pressed.
    • So, after the user presses the B button a melody plays letting the user know that the button has indeed been pressed and the function will run to show the compass location. This compass function uses looping structure to show the message run twice after being clicked by the user.
